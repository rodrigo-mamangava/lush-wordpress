<h4>Overview</h4>
Discounts are a powerful marketing weapon which allows you to drive customer loyalty and revenue. Woocommerce Advanced Discounts is a powerful extension which lets you manage shop discounts like a pro.

Woocommerce Advanced Discounts is built using WordPress best practices both on the front and the back end. This results in an efficient, robust and intuitive plugin. It's works with any theme, including the default WordPress themes and the last Woocommerce version.

<h4>Discount types</h4>
<ul>
	<li>Percentage based discounts</li>
	<li>Fixed amounts discounts</li>
</ul>

<h4>Discount conditions</h4>
<ul>
	<li>Customer role: Apply a discount if the customer has a specific role.</li>
	<li>Previous orders count: Apply a discount based on the number of previous orders the customer made.</li>
	<li>Order sub total: Apply a discount based on the order sub total.</li>
	<li>Order items count: Apply a discount based on the number of items in the cart.</li>
	<li>Products in cart: Apply a discount based on the products the customers is ordering (for example products in specific categories, products with specific tags or metas, products with specific attributes etc ..)</li>
</ul>

<h4>Pro features</h4>
<ul>
	<li><strong>Social networks integration</strong>: apply a discount if the customer is following at least one of the specified Facebook pages or instagram profiles</li>
	<li><strong>Payment Gateway</strong>: Apply a discount based on the payment gateway selected by the customer for the checkout process.</li>
	<li><strong>Billing country</strong>: Apply a discount based on the order or customer billing country.</li>
	<li><strong>Shipping country</strong>: Apply a discount based on the order shipping country.</li>
	<li><strong>Quantity pricing</strong>: Define a pricing table based on the purchased product quantity.</li>
	<li><strong>Groups based discounts</strong>: Apply a discount based on the users that belong in some particular groups. The groups are managed by the Groups plugin https://wordpress.org/plugins/groups/.</li>
	<li><strong>Newsletters based discounts</strong>: Apply a discount for all customers who subscribed to the newsletters managed by the Newsletter plugin https://wordpress.org/plugins/newsletter/.</li>
</ul>

<p><strong>This section describes how to install the plugin and get it working</strong></p>

<h4>Automatic installation (easiest way)</h4>

<p>To do an automatic install of Woocommerce Advanced Discounts, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.</p>

<p>In the search field type "Woocommerce Advanced Discounts" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".</p>

<h4>Manual installation</h4>

<p><strong>Uploading in WordPress Dashboard</strong></p>

<ol>
<li>Download <code>woocommerce-all-discounts.zip</code></li>
<li>Navigate to the 'Add New' in the plugins dashboard</li>
<li>Navigate to the 'Upload' area</li>
<li>Select <code>woocommerce-all-discounts.zip</code> from your computer</li>
<li>Click 'Install Now'</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p><strong>Using FTP</strong></p>

<ol>
<li>Download <code>woocommerce-all-discounts.zip</code></li>
<li>Extract the <code>woocommerce-all-discounts</code> directory to your computer</li>
<li>Upload the <code>woocommerce-all-discounts.zip</code> directory to the <code>/wp-content/plugins/</code> directory</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p>The WordPress codex contains <a href="http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation" rel="nofollow">instructions on how to install a WordPress plugin</a>.</p>

<h4>Updating</h4>

<p>You can use automatic update to update the plugin safely.</p></div>
